Flutter_Mall
==========

Flutter_Mall是一款Flutter开源在线商城应用程序，是基于[litemall](https://github.com/linlinjava/litemall)基础上进行开发，litemall包含了Spring Boot后端 + Vue管理员前端 + 微信小程序用户前端 + Vue用户移动端感兴趣的同学可以自行研究部署，Flutter_Mall基本上包含了litemall中小程序的功能。<br>

Flutter_Mall功能
----------
* 首页<br>
* 专题列表、专题详情<br>
* 分类列表、分类详情<br>
* 品牌列表、品牌详情<br>
* 新品首发、人气推荐<br>
* 优惠券列表、优惠券选择<br>
* 商品搜索<br>
* 商品详情<br>
* 购物车<br>
* 购物下单<br>
* 订单列表、订单详情<br>
* 地址、收藏、足迹、意见反馈<br>

项目结构
------------------
>lib
>>api(接口地址)<br>
>>config(路由相关)<br>
>>entity(实体类)<br>
>>event(事件)<br>
>>page(页面)<br>
>>service(网络请求)<br>
>>utils(工具类)<br>
>>widgets(通用组件)<br>

>>page(页面)<br>
>>>category(分类)<br>
>>>goods(商品)<br>
>>>login(登录注册)<br>
>>>mine(我的)<br>
>>>splash(启动页)<br>
